Lighting

I've done many lighting mods to the MR2 to improve the night look, as well as to improve efficiency as LEDs produce much more llight with less current as there is very little thermal waste compared to halogen globes. The basic mods include switching all interior and exterior lights to LEDs. This basically involved finding the globe socket type, and replacing with a matching LED. I did my best to colour match using a consistent temperature value for a similar white. The pop-up headlights, reverse lights and side markers were changed to LEDs by the previous owner. The basic swap I performed includes:

  • Parker Pights
  • Number plate Lights
  • Door Courtesy Lights
  • Gauge Cluster Lights
  • Trunk Lights
  • Rear Indicators

The larger and more interesting light projects I undertook were:

  • Quad Taillights
  • DRLs with Indicators
  • Underglow Neon
